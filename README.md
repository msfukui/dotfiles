# dotfiles

ふくい（ま）が使っている dotfiles です。

# Setting

## dotfiles setting.

```sh
$ cd ${HOME}
$ git clone https://github.com/msfukui/dotfiles.git
...
$ cd dotfiles
$ ./dotfiles.csh
...
$ ./dotfiles_diff.csh
...
```

## Vim setting.

```sh
$ cd ${HOME}
$ vim
...
:NeoBundleInstall
...
:q
$ vim
...
:q
```

## rbenv & ruby setting.

```sh
$ cd ${HOME}/dotfiles
$ ./rbenv_setup.sh
...
$ ./rbenv_ruby_setup.sh
...
```
